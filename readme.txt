This is a feature as a plugin demo for trac ticket 32396: The Settings Reduction Project

== Description ==

This is not just a plugin, it symbolizes the hope and enthusiasm of an entire core feature-as-a-plugin team summed up in a single plugin programmed most famously by Chris Christoff: Hello, World! When activated you will randomly see settings from ticket 32396 disappearing from your computer screen on every settings page.

See https://core.trac.wordpress.org/ticket/32396

== Installation ==

Just install like a standard WordPress plugin on a new 4.3 install or latest trunk
